## 问题 3: 内容被标注占据

**现状**：内容中包含大量：
- `采访素材仅能确认：...`
- `其余细节尚无直接证据`
- `（来源：素材 1）`

**修复方案**：
添加清理函数：
```python
def clean_text(text):
    import re
    if not text: return ""
    text = re.sub(r'采访素材 [^\n]*', '', text)
    text = re.sub(r'（来源：素材\d+）', '', text)
    text = re.sub(r'，其余细节尚无直接证据 [。.]?', '', text)
    text = re.sub(r'其余细节尚无直接证据', '', text)
    text = re.sub(r' +', ' ', text)
    return text.strip()
```
