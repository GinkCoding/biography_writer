### 2. 核心组件

#### ChapterGenerator（章节生成器）

功能：
- 根据规格生成单章内容
- 支持断点续传
- 分小节生成，降低单次生成压力
- 自动字数统计

使用：
```python
from src.generator import ChapterGenerator, ChapterSpec

gen = ChapterGenerator()
content = gen.generate_chapter(spec, source_material)
```

#### BookBuilder（书籍构建器）

功能：
- 合并所有章节
- 生成目录、序言、后记
- 输出统计报告

使用：
```python
from src.generator import BookBuilder

builder = BookBuilder()
output_file = builder.build_book(metadata, chapter_files)
```

### 3. 扩展到10万字的策略

#### 策略A：逐段扩写（推荐）

每章目标：20,000字
每章4节，每节约5,000字

扩写方法：
1. 原文：一句话概括事件
2. 扩写：添加以下细节
   - 环境描写（500字）
   - 人物外貌、动作、对话（1500字）
   - 心理活动（1000字）
   - 时代背景（1000字）
   - 感官细节（视、听、嗅、味、触）（1000字）

#### 策略B：批量AI生成

使用ChatGPT/Claude API，按以下提示词生成：

```
你是一位专业的传记作家。请将以下采访片段扩写为详细的传记章节。

原始素材：
[采访原文]

要求：
1. 字数：5000字
2. 包含：场景描写、对话、心理活动、时代背景
3. 风格：文学散文，细腻真实
4. 时间线：严格按照采访时间线

输出格式：
## 小节标题

[详细内容]
```

### 5. 技术实现方案

由于shell命令中的特殊字符问题，建议使用以下方式生成内容：

#### 方案1：Python脚本直接写入（已测试可行）

```python

content = """# 第三章

## 一、过关证

1988年，陈国伟做出了人生中最重要的一个决定..."""

with open("output/第三章.md", "w") as f:
    f.write(content)
```

#### 方案2：使用外部AI工具

1. 复制采访素材到ChatGPT/Claude

3. 保存生成的内容到文件

#### 方案3：分段生成

每次生成500-1000字，逐步累积到目标字数。
