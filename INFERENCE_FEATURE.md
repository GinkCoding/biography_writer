# 信息推理补全功能

## 功能概述

当采访信息较少时，系统会自动基于人物背景（出生年份、地域、性别、学历等）推断合理的中间过程，补足人物小传和大纲的空白内容。

## 核心原则

1. **合理性优先**：推断必须符合时代背景和社会规律
2. **可解释性**：每个推断都有明确的依据
3. **可追溯性**：推断内容明确标注为"合理推断"
4. **不犯错**：不编造具体事件，只推断类别和可能性

## 推断维度

### 1. 时代背景
系统内置中国近现代史关键节点数据库：
- 1949 新中国成立
- 1966-1976 文化大革命
- 1978 改革开放
- 1992 市场经济确立
- 2001 入世
- 等...

### 2. 教育轨迹推断
基于出生年代推断典型教育路径：
```python
# 1950s-1970s
["小学", "中学", "大学/技校", "分配工作"]

# 1960s-1970s（文革时期）
["小学", "中学", "文革中断", "上山下乡", "返城"]

# 1980s（高考恢复）
["小学", "中学", "高考", "大学", "分配/自主"]
```

### 3. 地域特征
- 东北：重工业基地 → 国企改革 → 产业转型
- 珠三角：改革开放前沿 → 制造业腾飞 → 科技创新
- 长三角：乡镇企业 → 浦东开发 → 金融中心
- 等...

### 4. 职业变迁规律
基于初始职业推断后续轨迹：
```python
"工人" + "1990s" → ["下岗", "个体户", "私企工人"]
"农民" + "1980s" → ["务农", "乡镇企业", "外出务工"]
"学生" + "1966-1976" → ["红卫兵", "上山下乡", "返城"]
```

### 大纲中的推断标记
推断的小节会被标记，并在生成时特殊处理：

```python
section = SectionOutline(
    id="inferred_1_gap",
    title="【推断】青年时期求学经历",
    target_words=500,
    content_summary="基于时代背景的合理推断...",
    is_inferred=True,  # 标记为推断内容
    inference_basis=["出生年份1965", "地域:北京", "时代:文革期间"]
)
```

## 生成时的特殊处理

对于推断内容，生成提示词会添加特殊约束：

```markdown
=== ⚠️ 重要提示：本节为推断内容 ===
本节内容基于人物的出生年份、地域背景、时代特征等信息进行合理推断...

写作要求：
1. 内容必须符合时代背景和社会规律，力求合理但不编造具体事件
2. 描述应为"类别性"而非"具体性"
3. 避免虚构具体人名、地名、数字，使用概括性表述
4. 可以描述典型的环境氛围、社会风气、常见的生活状态
```

## 信息缺口提示

系统会列出建议补充的信息：

```markdown
### 建议补充的信息
以下时间段信息不足，建议通过补充采访或资料核实：
- **青年时期信息空白** (high)
- **职业转换关键节点缺失** (medium)
- **家庭关系细节不足** (medium)
```

## 使用建议

1. **采访阶段**：尽可能获取关键时间点（入学、工作、婚嫁等）
2. **核查阶段**：对照推断内容，向传主核实或补充细节
3. **写作阶段**：推断内容可作为背景铺垫，具体事件仍需素材支撑

## 示例场景

### 场景1：采访信息较少
采访稿只提到：
- "我65年的，北京人"
- "后来去了工厂"
- "90年代下海做生意"

系统推断：
- 1965-1971 小学（文革期间）
- 1971-1977 中学（赶上恢复高考）
- 1977-1981 可能参加高考或工作
- 1981-1992 工厂工作（国企工人）
- 1992后 下海（南巡讲话后创业潮）

### 场景2：时间断档
采访稿有：
- 1960年出生
- 1978年考上大学
- 1995年成为教授

但缺少1980s信息：
系统推断：
- 1982-1985 研究生/博士
- 1985-1995 讲师/副教授（正常晋升路径）

## 注意事项

1. **推断不等于事实**：所有推断内容都需要核实
2. **个体差异**：群体规律可能不适用于个人
3. **敏感时期**：涉及敏感历史时期的推断需谨慎
4. **标记清晰**：推断内容在人物小传中明确标注
